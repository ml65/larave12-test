# Основная идея проекта

## Суть проекта

Мини-CRM система для автоматизации процесса сбора и обработки заявок от клиентов через универсальный виджет обратной связи, который может быть встроен на любой внешний сайт.

## Ключевая концепция

**Универсальный виджет обратной связи** — это основная точка входа для клиентов. Виджет реализован как отдельная Blade-страница, которая может быть встроена на любой сайт через `<iframe>`, что позволяет использовать систему на множестве различных платформ без необходимости интеграции API на стороне клиента.

## Целевые пользователи

1. **Клиенты** — используют виджет для отправки заявок с контактными данными, текстом обращения и прикрепленными файлами
2. **Менеджеры/Администраторы** — работают с заявками через админ-панель: просматривают, фильтруют, изменяют статусы, анализируют статистику

## Архитектурный подход

Проект построен на принципах чистой архитектуры:
- Разделение ответственности через сервисы и репозитории
- Строгая валидация через FormRequest
- RESTful API с использованием API Resources
- Следование принципам SOLID, MVC, KISS, DRY

## Технологический стек

- **Backend**: Laravel 12, PHP 8.4
- **База данных**: MySQL/PostgreSQL с Eloquent ORM
- **Файлы**: Spatie Laravel Media Library
- **Роли и права**: Spatie Laravel Permission
- **UI**: Blade шаблоны с современной стилизацией

## Уникальные особенности

- Виджет готов к использованию "из коробки" — достаточно встроить iframe
- Ограничение на создание заявок (не более одной в день с одного контакта) для предотвращения спама
- Полная статистика по заявкам с группировкой по периодам
- Многофайловые вложения к заявкам через медиа-библиотеку
